Iris classification¶

Classify Lily species based on features: sepal_length, sepal_width, petal_length, petal_width
using Supervised and Unsuperized approaches to classification

1) First run exploratiory data analysis to explore, get a feel for the data to inform modeling
2) Select and implement models and assess performance

Reference: http://archive.ics.uci.edu/ml/datasets/Iris
